Encounter Notes — The Table

Purpose

This opening encounter introduces The Table as a place of invitation
rather than instruction. The visitor should immediately feel welcomed
into a prepared place—a place where presence precedes explanation.

Use the standard encounter layout established by journey-01-guide.

Hero Image

Use the supplied table-01-web image as the full-screen background.

Do not crop in a way that loses the intimacy of the round table.
Preserve the balance between the prepared table and the open landscape
beyond. The table should remain the visual invitation.

Layout

Intent

The opening should not teach or explain.

It should create the feeling that the visitor has arrived somewhere
already prepared for them. The table is symbolic of welcome, presence,
belonging, and grace.

The encounter begins with invitation. Recognition comes later as the
visitor continues through the page.
